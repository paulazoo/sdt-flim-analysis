# sdt FLIM analysis

SDT (SPC Setup & Data File) FLIM analysis written in Python by Paula Zhu

for MCB68 class project

Largely inspired from examples:
- stdfile
  https://github.com/cgohlke/sdtfile
- flimview
  https://github.com/Biophotonics-COMI/flimview

## Requirements
- python3
- numpy
- matplotlib
- pandas
